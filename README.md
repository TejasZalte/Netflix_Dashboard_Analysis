 🎬 Netflix Data Analysis Dashboard | Power BI

 📌 Project Overview

This project presents a comprehensive analysis of Netflix's content library using Power BI. The dashboard provides valuable insights into movies and TV shows available on Netflix, helping users understand content distribution, trends, ratings, genres, release patterns, and global availability through interactive visualizations.

 🎯 Objectives

* Analyze Netflix's content catalog.
* Identify trends in movies and TV shows over time.
* Explore content distribution across countries and genres.
* Understand rating classifications and audience targeting.
* Create an interactive dashboard for data-driven storytelling.

 📊 Key Insights & Features

# Content Analysis

* Total Movies and TV Shows available on Netflix.
* Content distribution by type (Movies vs TV Shows).
* Growth of Netflix content over the years.
* Release year trends and content expansion.

### Genre Analysis

* Most popular genres on Netflix.
* Genre-wise content distribution.
* Comparison of genre popularity across content types.

# Geographic Analysis

* Top countries producing Netflix content.
* Country-wise content availability.
* Global content distribution visualization.

# Rating Analysis

* Distribution of content ratings (TV-MA, TV-14, PG, R, etc.).
* Audience segmentation based on content ratings.

# Time-Based Analysis

* Content added by year and month.
* Trends in Netflix's content acquisition strategy.
* Historical growth patterns.

 Interactive Dashboard Features

* Dynamic filters and slicers.
* Drill-down capabilities.
* Interactive visualizations and charts.
* User-friendly dashboard navigation.
* Real-time data exploration experience.

 🛠️ Tools & Technologies Used

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Cleaning & Transformation
* Interactive Dashboard Design

 📈 Business Value

This dashboard enables stakeholders to:

* Understand content trends and audience preferences.
* Identify high-performing genres and regions.
* Support content acquisition and recommendation strategies.
* Make data-driven decisions through visual analytics.

 📂 Dataset Information

The dataset contains information about Netflix Movies and TV Shows, including:

* Title
* Type
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genre (Listed In)
* Description
